# WP-03: Install Nerd Fonts

## Objective

Install Nerd Fonts on kovacs so that Doom Emacs icons, Starship prompt glyphs, and terminal UI elements render correctly.

## Deliverables

- Nerd Font(s) installed system-wide or per-user on kovacs
- Alacritty configured to use the Nerd Font
- Doom Emacs nerd-icons working

## Acceptance Criteria

- [ ] Starship prompt renders all glyphs correctly in Alacritty
- [ ] Doom Emacs shows icons in treemacs/modeline
- [ ] Font choice documented / added to molt-matts config

## Dependencies

- None — fully independent
